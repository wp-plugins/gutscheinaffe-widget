== Description ==

Das Gutscheinaffe Widget ist ein kostenloses Tool zum Anzeigen der Gutscheine von [Gutscheinaffe.de](http://www.gutscheinaffe.de/).
Dabei haben Sie vielfältige Möglichkeiten die Gutscheine auf Ihrem Wordpress Blog anzuzeigen.
Das Plugin funktioniert wie ein Widget in der Sidebar und lässt sich komplett farblich an Ihr Blogdesign anpassen.
Zusätzlich bieten wir Ihnen die Möglichkeit mit unseren Gutscheinen Geld zu verdienen. Sollten Sie Interesse daran haben, so kontakieren Sie uns doch bitte direkt über unsere Webseite.
Wir wünschen Ihnen ansonsten viel Spaß mit dem Plugin. Ihre Besucher werden sich sicherlich sehr über die neuen Sparmöglichkeiten auf Ihrem Blog freuen.
